# Claude Code / Orange Slice Integration Component

Landing-style component for connecting Claude Code to Orange Slice's prospecting, enrichment, and outreach tools.

## Placeholders

| Placeholder | Description | Example |
|-------------|-------------|---------|
| `PAGE_TITLE` | Browser tab title | Let's find your next customer — Claude Code |
| `HEADLINE` | Main page headline | Let's find your next customer. |
| `TAB_CHAT` | First tab label | Start a chat |
| `TAB_AGENT` | Second tab label | Build an agent |
| `TAB_CLAUDE_CODE` | Third tab label (selected) | Claude Code |
| `TERMINAL_TITLE` | Terminal window title | claude |
| `NPX_COMMAND` | The npx command (without `$`) | npx orangeslice@latest |
| `CONNECTION_MESSAGE` | Success message | Connected to Orange Slice |
| `TOOLS_LOADED` | Tools list | prospecting, enrichment, web_scraper, social_listening, outreach |
| `INSTRUCTIONS` | Explanatory paragraph (HTML allowed) | Run this command inside **Claude Code**... |
| `STEP_1` | First step text | Open Claude Code in your terminal |
| `STEP_2` | Second step text | Run the command above — it connects your session to Orange Slice |
| `STEP_3` | Third step text | Ask Claude to find leads, enrich data, or draft outreach |
| `LEARN_MORE_URL` | Link URL | https://orangeslice.ai |
| `LEARN_MORE_LABEL` | Link text | Learn more |

## Features

- Copy-to-clipboard for the npx command
- Terminal-style block with syntax highlighting
- Responsive layout using brand tokens
