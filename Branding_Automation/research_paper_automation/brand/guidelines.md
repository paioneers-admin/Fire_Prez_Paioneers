# Research Paper Brand Guidelines
**Paioneers B.V. — Research Division**

This document defines the typographic, layout, and visual standards for all Paioneers-branded research papers. These documents are intended for external audiences — customers, prospects, and industry peers — and must read as credible, peer-review-grade outputs.

---

## Design Philosophy

Research papers occupy a different register than pitch decks. The aesthetic goal is **academic authority**, not sales momentum. Every design choice should signal:

- **Rigour** — structured hierarchy, precise spacing, no decorative clutter
- **Credibility** — serif body text, formal tone, proper citation scaffolding
- **Paioneers identity** — visible but understated brand presence; we author this, we do not shout it

The brand palette is preserved but applied with restraint. The Paioneers orange (`#C96442`) and teal (`#0F7A99`) appear as structural accents — not dominant fills.

---

## Cover Page

The cover is the one place where brand is foregrounded:

- **Background:** Dark teal (`#032C2D`) — full bleed
- **Logo:** Top-left, white/cream version (`FAFFEF`)
- **Title:** Large, thin-weight serif (EB Garamond 28pt, weight 300), cream
- **Subtitle / Classification:** IBM Plex Mono, uppercase, wide-tracked (`0.12em`), muted cream
- **Authors / Date / Version:** Bottom-left block, small caps or mono, muted
- **Accent rule:** Single 1px horizontal line in teal (`#0F7A99`) separating title from metadata

No gradients, no images, no decorative fills on the cover beyond the solid background.

## Typography

### Body
- **Font:** EB Garamond (serif) — academic standard
- **Size:** 11pt
- **Line height:** 1.75
- **First-paragraph indent:** 1.5em (no indent after headings)
- **Colour:** Ink primary `#1A1A18`

### Headings
| Level | Style |
|---|---|
| H1 (Section) | IBM Plex Sans, 18pt, weight 500, ink primary, preceded by section number |
| H2 (Subsection) | IBM Plex Sans, 14pt, weight 500, ink secondary |
| H3 (Sub-subsection) | IBM Plex Sans, 12pt, weight 500, italic, ink secondary |
| H4 (Paragraph label) | IBM Plex Mono, 9pt, uppercase, wide-tracked, teal accent |

Section numbers (1., 1.1, 1.1.1) are always included. They use the mono font and muted ink.

### Captions
- IBM Plex Sans, 9pt, weight 400, ink muted (`#6B6B66`)
- Format: `Figure 1. Description.` or `Table 1. Description.`
- Positioned below figures, above tables

### Footnotes
- EB Garamond, 9pt, ink muted
- Separator: 1px rule, 30% page width, rule_light colour

## Layout

- **Paper size:** A4 (210×297mm)
- **Margins:** 25mm top/bottom; 30mm inner; 22mm outer
- **Column:** Single-column only (no multi-column layouts)
- **Max content width (HTML):** 720px centred
- **Paragraph spacing:** No extra space between body paragraphs — rely on indent
- **Section spacing:** 2.5em above each H1 section

### Page Header (running head)
- Left: Document title (abbreviated, ≤60 chars) — IBM Plex Mono, 8pt, muted
- Right: "Paioneers Research" — IBM Plex Mono, 8pt, muted
- Separated from body by 0.5px rule in `rule_light`

### Page Footer
- Centre: Page number — IBM Plex Mono, 8pt, muted
- Left: Version / DOI / Classification label (if applicable)

---

## Structural Elements

### Abstract
- Indented block (left+right 2em margin), 10pt, ink secondary
- Preceded by label: IBM Plex Mono "ABSTRACT", uppercase, teal accent, tracked
- Max 250 words

### Keywords
- Immediately below abstract
- Label: IBM Plex Mono "KEYWORDS" in teal accent
- Values: EB Garamond italic, comma-separated

### Callout / Key Finding Box
- Background: `#F0F4F2`
- Left border: 3px solid teal (`#0F7A99`)
- Padding: 16px 20px
- Label: IBM Plex Mono "KEY FINDING", teal, uppercase, 8pt
- Body: EB Garamond italic, 11pt

### Tables
- Header row: Dark teal background (`#032C2D`), cream text (`#FAFFEF`), IBM Plex Sans 10pt medium
- Alternating rows: white / `#F0F4F2`
- All borders: 0.5px `rule_light`
- No vertical borders between data columns (header only)

### Code / Data Blocks
- Background: `#F4F4F2`
- Font: IBM Plex Mono, 9pt
- Border: 1px `rule_light`
- Radius: 3px

### References
- Hanging indent (1.5em)
- EB Garamond, 10pt, ink secondary
- APA 7th format
- Section label: same H1 style

## Colour Usage Rules

| Element | Colour |
|---|---|
| Page background | `#FAFAF8` |
| Body text | `#1A1A18` |
| Secondary text / metadata | `#3D3D3A` |
| Muted text (captions, footnotes) | `#6B6B66` |
| Rules / dividers | `#D6D6D0` (light), `#A0A09A` (heavy) |
| Accent (teal) | `#0F7A99` — headings labels, callout borders, H4 |
| Accent (orange) | `#C96442` — hyperlinks, inline highlights only |
| Cover background | `#032C2D` |
| Cover text | `#FAFFEF` |

**Do not** use gradients, drop shadows, or full-bleed colour fills anywhere except the cover page.

---

## Logo Usage

- Cover page: White/cream logo, top-left, 120px wide maximum
- Running head: No logo — text identifier only
- Interior pages: No logo inline; appears only in page footer as a small wordmark (optional)

---

## What to Avoid

- Bullet lists as a primary content structure (use prose paragraphs)
- Bold used for decoration rather than genuine emphasis
- Tables without captions
- Unsourced claims or figures without attribution
- Mixing serif and sans-serif in body text
- Coloured backgrounds on any page except the cover
