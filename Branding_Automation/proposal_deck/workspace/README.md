This directory is for generating interactive Paioneers Proposals via Gamma.

# How to use

1. **Duplicate** the `_template_project/` folder and name it for your prospect (e.g., `Client_Acme`).
2. **Add Context:** Open `Client_Acme/reason.txt` and write the "Proposal Reason" (what are we pitching? what are their pains? pricing?).
3. **Add Screenshots:** Drop any relevant screenshots (LinkedIn profiles, company site, current bad processes) into `Client_Acme/screenshots/`.
4. **Generate:** Open this chat, ensure the Gamma MCP Server is active, and say:
   > "Generate the Gamma proposal deck for Client Acme."

I will automatically combined the reason, the vision analysis of the screenshots, and the Paioneers Brand CSS/Tokens to generate the exact presentation inside your Gamma account.
